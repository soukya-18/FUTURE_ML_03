# Resume Screening System
##project overview
This project builds a simple Resume Screening System using Machine Learning.  
It compares resumes with a job description and ranks candidates based on similarity.

## Objective
To automatically identify the best candidate for a job role using text similarity.

## Tools & Technologies
- Python
- Scikit-learn
- NumPy

## How It Works
1. Input resumes and job description
2. Convert text into numerical form using CountVectorizer
3. Calculate similarity using cosine similarity
4. Rank resumes based on similarity scores

## Output
- Similarity scores for each resume
- Ranking of candidates (best to worst)
- Best matching resume

##Key Learning
- Natural Language Processing (NLP)
- Text vectorization
- Cosine similarity
- Ranking systems

##Conclusion
The system successfully ranks resumes based on relevance to the job description, helping in faster and smarter hiring decisions.
